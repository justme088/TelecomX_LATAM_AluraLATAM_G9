Este proyecto presenta un **análisis exploratorio de datos (EDA)** sobre la evasión de clientes (*Churn*) en una empresa de telecomunicaciones. El objetivo es identificar los factores que influyen en que los clientes abandonen el servicio y generar insights que ayuden a mejorar la **retención de clientes**.

El análisis se realizó sobre un dataset de **7,043 clientes**, evaluando variables como antigüedad del cliente, tipo de contrato, cargos mensuales, servicios contratados y características demográficas.

---

## 🎯 Objetivo

Identificar patrones asociados al **abandono de clientes** y comprender qué variables tienen mayor impacto en la probabilidad de churn.

---

## 🔎 Análisis realizado

Durante el proyecto se analizaron diferentes aspectos del comportamiento de los clientes:

- Antigüedad del cliente (**Tenure**)
- Cargos mensuales y gasto del cliente
- Tipo de contrato
- Servicios de internet (DSL, Fiber Optic)
- Variables demográficas
- Correlaciones entre variables

Se utilizaron técnicas de **análisis exploratorio de datos y visualización** para detectar patrones relevantes.

---

## 📈 Principales hallazgos

- La evasión ocurre principalmente en **clientes con baja antigüedad**.
- Los **contratos mes a mes** presentan mayor tasa de cancelación.
- Los clientes con **cargos mensuales más altos** tienen mayor probabilidad de churn.
- El servicio de **fibra óptica** concentra una gran parte de las cancelaciones.
